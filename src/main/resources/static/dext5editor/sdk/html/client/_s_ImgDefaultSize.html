DEXT5 Editor :: ImgDefaultSize

XML : <img_default_size width="">.....</img_default_size>, Config : config.ImgDefaultWidth
XML : <img_default_size height="">.....</img_default_size>, Config : config.ImgDefaultHeight

이미지 너비, 높이 기본값을 설정합니다.

remarks

예) width="100" 설정 시
너비는 "100" / 높이는 너비의 값에 비례하여 적용됩니다.

sample code

이미지 너비 기본값을 200으로 설정합니다.

XML

.....
<setting>
    .....
    <img_default_size width="200" height=""></img_default_size>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

        DEXT5.config.ImgDefaultWidth = "200";
        DEXT5.config.ImgDefaultHeight = "";

        new Dext5editor("editor1");

    </script>       
</div>
